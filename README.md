# Flutter News App

Flutter News App

## Getting Started

Flutter News Application using getx provider
# Features 
- Cloud Cache 
- Theme Change
- API Integration

# Build Process
Get your api key from https://newsapi.org and add it inside .env with API_KEY=Your Key as shown in .env.example
## Debug 
flutter run --debug
## Profile 
flutter run --profile
## Profile 
flutter run --release
